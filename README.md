## How to Contribute

1. Fork this repository
2. Create your branch: `git checkout -b feature/member-X-task-name`
3. Make your changes in your assigned file and section
4. Commit: `git commit -m "Member X: Task description"`
5. Push: `git push origin feature/member-X-task-name`
6. Open a Pull Request on GitHub

See the full step-by-step guide in `CONTRIBUTING.md`
